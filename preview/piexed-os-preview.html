Key Features

Everything you need, optimized for performance

⚡

Lightweight & Fast

Optimized for systems with just 1GB RAM. Uses zram compression, earlyOOM, and smart caching to deliver smooth performance.

💻

Ubuntu Compatible

Full support for all Ubuntu packages. Install anything from the vast Ubuntu ecosystem with apt, snap, or flatpak.

🌐

macOS Experience

Familiar dock with magnification, top panel with window controls, mission control workspaces, and smooth animations.

📱

App Store

Modern GTK3-based software center. Browse, install, and update applications with one click.

🔋

Dual Boot Ready

Automatically detects Windows and other Linux installations. Install alongside your existing OS without data loss.

🚧

Android Support

Run on Android devices via Termux or UserLAnd. Full Linux environment on your mobile device.
